# Ottawa Paramedic Demand Analysis (2021–2023)

## Project Overview
This project analyzes daily paramedic service demand in Ottawa from 2021 to 2023 using Power BI.

The goal of the analysis is to understand:
- how demand changes over time
- which days show higher demand
- whether there are seasonal patterns
- how growth trends are changing year over year

## Tools Used
- Power BI
- Excel / CSV
- DAX

## Dataset
- Ottawa Paramedic Service daily response volume dataset

## Key Insights
- Paramedic demand increased over time from 2021 to 2023.
- Fridays showed the highest average demand across the week.
- Demand varied across months, suggesting seasonal patterns.
- Total demand continued to grow, but the growth rate slowed over time.
